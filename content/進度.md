## 2025/08/22

- 複習 function
- 複習 for loop

  ```python
  for i in range(len()):
  ```

  > `range()` 和 `len()` 都是 function

- 複習 `range()`

  ```python
  # range() 可以有 1~3 個參數

  # 1 個參數
  range(5)
  # 等同於
  range(0, 5)
  range(start=0, stop=5)

  # 2 個參數
  range(0, 5)
  # 等同於
  range(start=0, stop=5)
  # 3 個參數
  range(0, 5, 1)
  # 等同於
  range(start=0, stop=5, step=1)
  ```

- 複習 `len()`
