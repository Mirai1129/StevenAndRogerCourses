3.  實作一個根據用電度數，計算出電費的簡單程式。

    - 電費計算規則如下：

      - 電度數 0 ~ 200 度，每度 3.2 元。
      - 電度數 201 ~ 300 度，每度 3.4 元。
      - 電度數大於 300 度，每度 3.6 元。

    - 輸入輸出如下：

      ```
      > 請輸入用電度數：30
      用電度數 30.0 度，電費為：96.0 元
      > 請輸入用電度數：201
      用電度數 201.0 度，電費為：643.4 元
      > 請輸入用電度數：400
      用電度數 400.0 度，電費為：1340.0 元
      ```

    - 應該要用到的：

      - while loop
      - if-else 條件判斷
      - input() 取得用電度數
      - print() 顯示電費 -> fstring

    - 範例程式碼：
      ```python
      while True:
          try:
              num = float(input('請輸入用電度數：'))
              output = 0
              if num <= 200:
                  output = num * 3.2
              elif num > 200 and num <= 300:
                  output = 200 * 3.2 + (num - 200) * 3.4
              else:
                  output = 200 * 3.2 + 100 * 3.4 + (num - 300) * 3.6
              print(f'用電 {num} 度共 {output} 元')
          except:
              break
      ```

### 數學 & 邏輯推理

1. 剛好一個是偶數

   目標：練習邏輯判斷與計數。

   - 題目：

     - 讀入三個整數 a, b, c。
     - 判斷三個數中是否「剛好一個是偶數」，輸出 True 或 False。

   - 範例：

     ```
      輸入：2 3 5 -> 輸出：True
      輸入：2 4 6 -> 輸出：False
      輸入：1 3 5 -> 輸出：False
     ```

   - 提示：

     - 用 n % 2 == 0 判斷偶數，把偶數個數加起來，看是否等於 1。

   - 進階挑戰：

     - 改成判斷「剛好兩個是偶數」。

2. 找捷徑：1 到 n 的總和

   目標：發現規律與比較做法。

   - 題目：

     - 讀入一個整數 n。
     - 用「迴圈」算 1 + 2 + ... + n。
     - 再用「公式」`n*(n+1)//2` 算一次。
     - 輸出兩種方法的結果。

   - 範例：

     ```
       n = 5 -> 迴圈=15，公式=15
       n = 100 -> 迴圈=5050，公式=5050
     ```

   - 提示：

     - 觀察兩種方法哪個更快（n 很大時）。

   - 進階挑戰：

     - 對很大的 n（例如 1_000_000）只印出公式結果，思考為什麼。

3. 比大小：三數排序

   > 這題和前面的 `loop and basic operation` 題目類似。

   目標：理解比較與排序。

   - 進階挑戰：

     - 印出中位數（中間那個數字）。

7. 水仙花數 (Narcissistic number)

   - 題目：

     - 讀入一個三位數整數 n。
     - 判斷 n 是否為水仙花數（每個位數的立方和等於 n 本身）。
     - 輸出 True 或 False。

   - 範例：

     ```
     數字： 153 (3 digits)
     1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153

     數字： 1634 (4 digits)
     1^4 + 6^4 + 3^4 + 4^4 = 1 + 1296 + 81 + 256 = 1634
     ```

   - 提示：

     - 可以用 `n // 100` 取得百位數，`n // 10 % 10` 取得十位數，`n % 10` 取得個位數。

   - 進階挑戰：

     - 改成判斷任意位數的水仙花數。
